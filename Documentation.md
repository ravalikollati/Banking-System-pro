## Project Name

Banking System Console Project

## Project Overview

Banking System Console Project is a web-based application developed using HTML, CSS, and JavaScript. The system enables users to create customer accounts, manage account balances, perform deposits, and monitor banking statistics through an interactive dashboard.

## Project Scope

The project aims to provide a simple banking management solution that allows users to maintain customer account information and perform basic banking operations efficiently through a user-friendly interface.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

### 1. Account Creation

Users can create new customer accounts by entering:

* Customer Name
* Account Number
* Initial Deposit Amount

### 2. Dashboard Statistics

The system automatically displays:

* Total Number of Accounts
* Total Bank Balance
* Richest Customer

### 3. Account Management

All customer records are displayed in a structured table containing:

* Customer Name
* Account Number
* Current Balance

### 4. Deposit Functionality

Users can deposit ₹1000 into any account using the Deposit button available in each row.

### 5. Dynamic Updates

The dashboard statistics update automatically whenever account information changes.

## Working Procedure

1. Enter customer details in the input fields.
2. Click the "Create Account" button.
3. The account is added to the banking system.
4. Customer information appears in the records table.
5. Use the Deposit button to add funds to a customer's account.
6. The system updates the total balance and richest customer information automatically.

## Output Description

The output screen contains:
