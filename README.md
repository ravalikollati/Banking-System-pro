Banking System Console

## Project Scope

The Banking Management System is a web-based application developed using HTML, CSS, and JavaScript. It provides a simple interface for managing banking operations such as customer account management, deposits, withdrawals, balance tracking, and transaction handling.

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Features

* Create Customer Accounts
* Deposit Money
* Withdraw Money
* View Account Details
* Balance Management
* User-Friendly Interface
* Responsive Design
